# Machine Learning

This is a simple maze game where a Machine learns to escape quickly.

```ascii
+-------------+
|      u      |
|    l A r    |
|      d      |
|             |
```

The grid [x,y] contains an action object with change indicators for direction.

action.up = 0.25
action.down = 0.25
action.left = 0.25
action.right = 0.25
